<h1>8-bit Processor Design</h1>
<p><strong>Course:</strong> Computer Architecture and Organization </p>

<div class="card">
  <p>
    This project implements a simple <strong>8-bit processor</strong> designed using
    logic gates and atomic digital components. The processor architecture follows
    the requirements provided in the course and is fully simulated using
    <strong>Logisim</strong>.
  </p>
</div>

<h2>Architecture Summary</h2>
<ul>
  <li>8-bit data path and registers</li>
  <li>Hardwired control unit</li>
  <li>Gate-level ALU design</li>
</ul>

<h2>ALU Operations</h2>
<ul>
  <li>Arithmetic: ADD, SUB, MUL, DIV</li>
  <li>Logical: XOR, NAND, XNOR, NOR</li>
</ul>

<h2>Instruction Support</h2>
<ul>
  <li>8 instruction opcodes</li>
  <li>One-address and two-address formats</li>
  <li>Direct and indirect addressing modes</li>
</ul>

<h2>Registers</h2>
<ul>
  <li>PC, IR, MAR, MBR</li>
  <li>Additional internal registers as required</li>
</ul>

<h2>Simulation</h2>
<ul>
  <li>Designed and tested using Logisim</li>
  <li>Instruction execution and ALU operations verified</li>
</ul>
